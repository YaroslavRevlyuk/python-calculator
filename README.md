# Графический калькулятор на Python

Простой калькулятор с графическим интерфейсом, реализованный на Python с использованием библиотеки Tkinter.

## Функциональность

-  Базовые арифметические операции (+, -, *, /)
-  Возведение в степень (^) 
-  Работа с десятичными дробями
-  Отрицательные числа
-  Управление с клавиатуры
-  Обработка ошибок ввода

##  Технологии

- Python 3
- Tkinter (для графического интерфейса)
- Регулярные выражения (для парсинга ввода)
